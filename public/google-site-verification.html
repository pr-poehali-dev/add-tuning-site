Google Site Verification

Этот файл используется для верификации сайта в Google Search Console.

После верификации вы сможете:

  • Отслеживать позиции в поиске Google
  • Видеть поисковые запросы пользователей
  • Получать уведомления о проблемах индексации
  • Отправлять sitemap для быстрой индексации
